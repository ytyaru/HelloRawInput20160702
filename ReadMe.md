# このソフトウェアについて #

HelloRawInput20160702は私個人が学習目的で作成したソフトウェアです。

複数のマウスを検出します。
押下したボタンなどを判別できます。

Visual C++ 2010 Expressのデバッグ出力で確認できます。
Windowには何も表示されません。

# 開発環境 #

* Windows XP SP3
* C++/Win32
* Visual C++ 2010 Express
